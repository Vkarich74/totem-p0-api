# OWNER_AUDIT_LOG

Purpose:
Track owner/admin actions for accountability.

---

## Logged actions

- booking cancel
- booking complete
- no_show mark
- staff changes
- settings changes

---

## Log fields

- actor_id
- role
- action
- target_id
- timestamp

---

## Rules

- Logs are append-only
- Logs are not editable
- Used for dispute resolution

End of document.
